Automorph is an RPC client and server library for Scala providing an effortless way to invoke and expose remote APIs using JSON-RPC and REST-RPC protocols.

Integration

Automorph supports integration with various libraries via plugins published in different artifacts.

Effect system

Computational effect system plugins.

The underlying runtime must support monadic composition of effectful values.

Class Artifact Library Effect Type
FutureSystem (Default) automorph-standard Standard Future
TrySystem automorph-standard Standard Try
IdentitySystem automorph-standard Standard Identity
ZioSystem automorph-zio ZIO RIO
MonixSystem automorph-monix Monix Task
CatsEffectSystem automorph-cats-effect Cats Effect IO
ScalazSystem automorph-scalaz Scalaz IO

Message format

Structured message format serialization/deserialization plugins.

The underlying format must support storing arbitrarily nested structures of basic data types.

Class Artifact Library Node Type Format
CirceJsonFormat (Default) automorph-circe Circe Json JSON
UpickleJsonFormat automorph-upickle uPickle Value JSON
UpickleMessagePackFormat automorph-upickle uPickle Msg MessagePack
ArgonautJsonFormat automorph-argonaut Argonaut Json JSON

Message transport

Message transport protocol plugins.

The underlying transport protocol must support request/response messaging pattern.

Client transport

Client message transport protocol plugins.

Used by the RPC client to send requests and receive responses to and from a remote endpoint.

Class Artifact Library Effect Type Protocol
SttpClient (Default) automorph-sttp STTP -> Akka HTTP, AsyncHttpClient, HttpClient, OkHttp All HTTP, WebSocket
HttpUrlConnectionClient automorph-standard Standard Identity HTTP
RabbitMqClient automorph-rabbitmq RabbitMq Future AMQP

Server transport

Server message transport protocol plugins.

Used to actively receive and reply to requests using specific message transport protocol while invoking RPC request handler to process them.

Class Artifact Library Effect Type Protocol
UndertowServer (Default) automorph-undertow Undertow All HTTP, WebSocket
NanoHttpdServer automorph-standard NanoHTTPD All HTTP
RabbitMqServer automorph-rabbitmq RabbitMq All AMQP

Endpoint transport

Endpoint message transport protocol plugins.

Used to passively receive and reply to requests using specific message transport protocol from an active server while invoking RPC request handler to process them.

Class Artifact Library Effect Type Protocol
UndertowHttpEndpoint automorph-undertow Undertow All HTTP
UndertowWebSocketEndpoint automorph-undertow Undertow All WebSocket
TapirHttpEndpoint automorph-tapir Tapir -> Akka HTTP, Finatra, http4s, Play, Vert.X, ZIO Http All HTTP
JettyEndpoint automorph-jetty Jetty All HTTP
FinagleEndpoint automorph-finagle Finagle All HTTP

Client

The client provides automatic creation of transparent proxy instances for remote RPC endpoints defined by existing API classes. Additionally, it also supports direct calls and notifications of remote API methods.
